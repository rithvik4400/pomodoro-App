# ⏳ Zen Timer

A beautiful, minimalist Pomodoro timer built with vanilla web technologies. Zen Timer helps you stay focused and calm using the Pomodoro Technique, featuring smooth animations, a glassmorphic design, and audio notifications.

## ✨ Features

- **Three Timer Modes:**
  - 🍅 **Pomodoro** (25 minutes): Deep work session.
  - ☕ **Short Break** (5 minutes): Quick breather to stretch and relax.
  - 🛋️ **Long Break** (15 minutes): Extended rest after completing multiple Pomodoros.
- **Dynamic Interface:** Background themes automatically transition based on the active mode to set the right mood.
- **Visual Progress Tracking:** An elegant, animated SVG circular progress bar shows exactly how much time is left.
- **Audio Notifications:** A subtle chime plays when your session or break is complete.
- **Browser Tab Updates:** The remaining time is displayed right in the browser tab title so you can track it while working in other windows.
- **Modern Aesthetics:** Clean, responsive design featuring glassmorphism elements, custom typography (Inter), and premium imagery.

## 🚀 Tech Stack

- **HTML5:** Semantic structure and integrated SVG graphics.
- **CSS3:** 
  - Custom variables for seamless theme switching.
  - Flexbox/Grid for responsive layout.
  - Advanced styling including glassmorphism and smooth CSS transitions.
- **Vanilla JavaScript:** 
  - DOM manipulation and event handling.
  - Precise interval-based timer logic.
  - HTML5 Audio API for sound playback.
  - *Zero external dependencies or frameworks.*

## 💻 Usage

Running the application is incredibly simple. Since it uses vanilla web tech, no complex build process is required!

1. Clone or download the source files.
2. Navigate to the project directory (containing `index.html`, `style.css`, and `script.js`).
3. Simply double-click on `index.html` to open it in your preferred web browser (e.g., Chrome, Firefox, Safari, Edge).
4. Select your mode (Pomodoro, Short Break, or Long Break) and click **Start**!

*Note: Some modern browsers restrict audio playback unless the user has actively interacted with the page. You shouldn't have any issues since you must click "Start" to begin the timer.*

## 📖 How It Works

The Pomodoro Technique is a time management method developed by Francesco Cirillo in the late 1980s. The process is simple:

1. Pick a task you want to work on.
2. Start a **Pomodoro** (25 minutes) and work continuously until the timer rings.
3. Take a **Short Break** (5 minutes) to relax your mind.
4. Every four Pomodoros, take a **Long Break** (15-30 minutes).

Stay focused, stay calm.
